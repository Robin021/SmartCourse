# 一、产品概述与战略目标

# （一）产品定位

以“教育AI咨询+课程生成”为双引擎的SaaS产品，面向K12学校及教育管理部门，提供从课程顶层设计、教学内容生成到学情评估的全流程智能化解决方案，实现教育理念与实践的数字化落地。

# （二）愿景与使命

1. 愿景：成为教育智能化转型的核心驱动力，让每所学校都能拥有个性化优质课程体系  
2. 使命：通过AI技术链接教育哲学、校情特色与教学实践，推动从课程数字化到教育智能化的跨越

# （三）产品核心价值与差异化

1. 双引擎协同：AI咨询模块提供专业诊断（如SWOT量化分析），课程生成模块落地实践方案，形成“诊断-设计-实施-评估”闭环  
2. 在地化适配：深度融合地域文化（如运河文化）、校情特色（如红色基因）与政策要求，生成专属课程体系  
3. 全流程共创：教师可通过对话式交互参与课程设计全环节，AI动态响应修改需求，平衡智能化与个性化  
4. 数据驱动优化：通过课程评估雷达图、学情分析等可视化工具，实现课程持续迭代

# （二）典型使用场景

# 1. 课程顶层设计场景

1. 校长输入学校SWOT数据、教育哲学倾向、地域文化等信息  
2. AI生成《学校教育哲学陈述》《办学理念阐释》《课程结构方案》等系列文档  
3. 教研主任补充学段细节，教师团队在线协作修改，形成最终方案

# 1. AI诊断报告场景

1. 学校上传现有课程体系与学情数据  
2. AI通过RAG检索政策标准与标杆案例，诊断课程目标与育人目标的差距点  
3. 输出含 SWOT 量化分析、薄弱维度预警的诊断报告，附优化建议

# 1. 区域对比场景

1. 教育局导入区域内多所学校课程数据  
2. AI 聚合分析各校课程特色、五育覆盖度、目标达成率等指标  
3. 生成区域课程质量雷达图，识别优势领域与改进方向

# （三）核心功能说明

# 1. AI共创工作区

1. 对话式交互：教师以自然语言提出修改需求（如“增加运河文化实践模块”），AI实时迭代内容  
2. 版本管理：自动保存每轮修改记录，支持回溯对比（如“恢复Q3.2核心概念初稿”）  
3. 实时协作：多人在线标注文档，显示修改者身份与意见，支持投票决策

# 2. 课程设计生成和导出

1. 生成格式：支持Word（含标准教育文档排版）、PDF、PPT（可视化结构图）  
2. 导出选项：全流程打包导出（十阶段成果）、单阶段导出、自定义组合导出  
3. 模板适配：内置教育局备案模板、校本科研模板等3类标准模板

# 3. 课堂教学内容生成和导出

1. 生成内容：基于课程目标的教案、课件、探究任务单、作业设计  
2. 学段适配：自动匹配低/中/高学段认知水平（如低年级增加图文比例）

3. 导出功能：支持分课时导出、按学科分类导出，课件兼容PPT/Keynote格式

# 4. 教师辅助工具集

1. 课程规划工具：输入主题自动生成跨学科教学序列（如“运河文化”  $\rightarrow$  语文  $+$  历史  $+$  科学融合方案）  
2. 学情分析工具：上传作业/测试数据，生成学生能力雷达图与个性化辅导建议  
3. 资源匹配工具：根据课程模块推荐优质素材（如视频、实验手册、外部场馆资源）  
4. 作业生成工具：按知识点生成分层作业，支持自动批改客观题

# 5. 学生成长平台

1. 成长档案：记录课程参与数据、评价结果、获奖情况，生成电子成长手册  
2. 目标追踪：可视化展示学生与育人目标的差距（如“数码创造力”达标进度）  
3. 互动功能：学生上传实践成果（如项目报告、手工作品），教师在线点评

# 6. 课程评估与雷达分析仪表板

1. 核心指标：五育覆盖度、目标达成率、学生参与度、教师实施难度  
2. 可视化呈现：雷达图（校际对比）、趋势图（月度/学期变化）、热力图（模块优势分布）  
3. 预警功能：指标低于阈值时自动提示（如“德育模块达成率低于  $60\%$ ”）

# 四、关键前端页面交互逻辑设计

# （一）登录页

1. 进入页面：显示产品Logo与“教育AI共创平台”标语，支持账号密码登录与验证码登录  
2. 角色选择：登录后弹出角色选择框（校长/教研主任/教师/教育局），不同角色跳转至对应工作台  
3. 权限验证：系统校验角色权限，如教师端隐藏区域对比功能  
4. 记住登录：提供“记住账号”选项（默认7天），支持“忘记密码”（通过绑定手机号重置）

# （二）十阶段课程设计输入和内容生成页

1. 阶段导航：顶部显示十阶段进度条（如“Q1学校课程情境[已完成]  $\rightarrow$  Q2教育哲学[进行中]”），支持直接跳转  
2. 输入区域：

1. 选择题：弹出下拉列表（如Q2哲学理论选择），可多选，实时显示已选项  
2. 开放式输入：提供文本框（支持换行/粘贴），Q1SWOT表格采用可编辑表格组件（内外维度分栏）  
3. 补充说明：每个输入项旁有“示例”按钮，点击显示参考案例（如Q3.1价值观问题示例)

3. 生成交互：点击“生成内容”按钮后显示加载动画，生成完成后展示结果面板（左侧输入/右侧输出）  
4. 修改交互：输出面板提供“修改建议”文本框，输入需求后点击“迭代生成”，新增内容标红显示

# （三）AI共创工作区

1. 布局设计：左侧对话区（显示历史交互记录），右侧内容编辑区（当前文档）  
2. 对话交互：顶部搜索框输入需求（如“优化Q6.3特色目标”），AI回复显示在对话区，支持追问（如“增加劳育相关描述”）  
3. 编辑功能：内容编辑区支持富文本操作（加粗、插入表格/图片），点击“应用AI建议”自动更新文档  
4. 协作提示：多人协作时，他人修改内容以蓝色高亮显示，弹出“XXX正在编辑”提示

# （四）课程设计生成和导出页

1. 成果预览：左侧显示十阶段成果列表，点击某阶段右侧显示预览内容（支持翻页）  
2. 导出设置：顶部选择导出格式（Word/PDF/PPT）、导出范围（全选/自定义勾选）、模板类型  
3. 操作反馈：点击“开始导出”后显示进度条，完成后弹出“导出成功”提示，支持“立即下载”/“发送至邮箱”

# （五）课堂教学内容生成和导出页

1. 课程关联：自动显示当前关联的课程目标（如“Q7.2 智慧维度高年级目标”），支持切换关联目标  
2. 内容选择：勾选需生成的内容类型（教案/课件/作业），点击“生成”后显示内容列表  
3. 个性化设置：提供“难度调整”滑块（低/中/高）、“素材类型”选项（图文/视频/实验）  
4. 导出交互：同课程设计导出，支持按课时批量导出

# （六）教师辅助工具页

1. 工具导航：左侧分类显示工具列表（课程规划/学情分析/资源匹配/作业生成），点击切换  
2. 操作流程：

1. 选择工具  $\rightarrow$  输入参数（如“作业生成”需选择知识点/难度）  $\rightarrow$  点击生成  $\rightarrow$  预览修改  $\rightarrow$  导出

1. 结果展示：学情分析工具生成雷达图，支持点击指标查看详细说明（如“逻辑推理能力薄弱原因”）

# （七）课程评估与雷达分析仪表板

1. 数据筛选：顶部选择时间范围（月度/学期/学年）、对比对象（本校vs区域平均/标杆校）  
2. 图表交互：

1. 雷达图：鼠标悬停显示具体数值（如“美育覆盖度  $85\%$ ”）  
2. 趋势图：支持缩放查看细节，点击节点显示对应时期改进建议

3. 报告生成：点击“生成评估报告”，自动整合图表与分析文字，支持导出Word格式
